## Why use-legend?

[VueUse](https://vueuse.org/) and [react-use](https://github.com/streamich/react-use) proved that a composable utility collection can dramatically improve developer experience. `use-legend` brings that same philosophy to the observable world — every hook is designed from the ground up to work with [Legend-State](https://legendapp.com/open-source/state/) observables, delivering fine-grained reactivity instead of whole-component re-renders.
