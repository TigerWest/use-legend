Platform-agnostic API reference for foundational hooks.

## Installation

- Web: `npm install @usels/web@beta @legendapp/state react`
- Native: `npm install @usels/native@beta @legendapp/state react react-native`

Core hooks are included in both platform packages.  
Install either `@usels/web` or `@usels/native` based on your app target.
